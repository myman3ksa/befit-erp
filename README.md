# MAN-3 Plus ERP System v1.2

A comprehensive, cloud-based Restaurant ERP System built for high-performance multi-branch operations.

## 🚀 Key Modules
- **Purchasing**: Full P.O. cycle with VAT auto-calculation and 3-step approval flow.
- **Inventory**: Manual item/category management and stock tracking.
- **Production**: Real-time yield calculation and ingredient deduction engine.
- **Finance**: Supplier Statement of Account (SOA), Invoice tracking, and Urgent Payment priority system.
- **Branch Management**: Centralized control for adding/editing branch locations.
- **Dashboard Analytics**: KPI-driven insights with Chart.js visualizations.
- **Kitchen Dashboard**: Daily production and ingredient monitoring.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Backend**: [Supabase](https://supabase.com/) (PostgreSQL + Auth).
- **Charts**: [Chart.js](https://www.chartjs.org/).
- **Icons**: [Boxicons](https://boxicons.com/).

## 📦 Features
- **Multi-Branch Support**: Filters and comparisons between branches.
- **Dual-Language**: Full English and Arabic (KSA) support with RTL.
- **Reporting**: Export any section (Purchasing, Finance, Production) to **PDF** or **XLS**.
- **Dark Mode**: Premium, modern dark-themed user interface.

## 🚦 Approval Workflow (Purchasing)
1. **Procurement Officer**: Initiates P.O.
2. **Procurement Assistant**: Reviews and verifies items.
3. **Finance Department**: Final approval for payment.

## 👨‍💻 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/myman3ksa/befit-erp.git
   ```
2. Configure your Supabase credentials in `app.js`.
3. Open `index.html` in any modern browser or host via GitHub Pages.
